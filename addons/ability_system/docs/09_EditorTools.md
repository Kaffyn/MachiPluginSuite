# Editor Tools — Workbench

O plugin vem com um painel inferior dedicado.

## 1. Behavior Library

Explore e filtre todos os States, Composes e Componets do projeto.

- Visualização em Grid ou Lista.
- Filtros por Tags.

## 2. Factory

Interface rápida para criar novos arquivos `.tres`.

- Templates pré-configurados.
- Evita esquecer de preencher o `script`.

## 3. Debugger

Visualize o cérebro da AI em tempo real.

- **Mostra:** Contexto atual, Estado ativo, Score dos candidatos.
- **Histórico:** Log das últimas transições (útil para entender loops ou flickers).

## 4. Visual Compose Editor (Futuro)

Planejado: Arrastar States para um Canvas para visualizar conexões lógicas (Combos).
